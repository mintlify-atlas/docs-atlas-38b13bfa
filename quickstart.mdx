<Warning>
  Using self-bots violates Discord's Terms of Service. Deleo accesses the Discord API using your personal user token. Your account may be suspended or banned. Use at your own risk.
</Warning>

Follow these steps to authenticate, select channels, and start deleting your Discord messages.

## Prerequisites

- [Bun](https://bun.sh/) installed on your system
- Windows (primary supported platform)
- Your Discord token (see [Token Helper](/token-helper/overview) or [Manual Extraction](/token-helper/manual-extraction))

## Steps

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/kars1996/deleo.git
    cd deleo
    bun install
    ```
  </Step>

  <Step title="Get your Discord token">
    You need your Discord user token to authenticate. There are two ways to get it:

    <Tabs>
      <Tab title="Auto-detect (recommended)">
        Use the included Python script to extract your token automatically from your browser or Discord client:

        ```bash
        cd scripts
        pip install -r requirements.txt
        python token-helper.py --save
        ```

        This saves your token to `~/.deleo_cached_token`. Deleo will load it automatically on the next launch.

        <Note>
          The token helper is Windows-only. See [Token Helper](/token-helper/overview) for full details.
        </Note>
      </Tab>
      <Tab title="Manual extraction">
        1. Open Discord in your browser
        2. Open DevTools with **F12**
        3. Go to the **Network** tab
        4. Send any message in any channel
        5. Click the request that appears and find the `authorization` header
        6. Copy the token value

        You will paste this into Deleo in the next step.
      </Tab>
    </Tabs>
  </Step>

  <Step title="Run Deleo">
    ```bash
    bun run dev
    ```

    Electrobun opens a native window with the Deleo UI. You should see the authentication screen.
  </Step>

  <Step title="Authenticate">
    On the authentication screen:

    - **Auto-detect** — click this if you used `token-helper.py --save`. Deleo loads your token from `~/.deleo_cached_token` and validates it against the Discord API.
    - **Enter manually** — paste your token directly. You can choose to save it for future sessions.

    When authentication succeeds, your Discord username and avatar appear and the app moves to channel selection.
  </Step>

  <Step title="Select channels">
    Deleo fetches your DM channels and group chats. Select one or more channels whose messages you want to delete.

    <Tip>
      You can preview how many messages you have in each channel before starting deletion.
    </Tip>
  </Step>

  <Step title="Configure deletion speed">
    Set the **deletion delay** — the wait time between each deleted message. The default is 1000ms (1 second). Lower values delete faster but increase rate limit risk.

    | Delay | Speed | Risk |
    |-------|-------|------|
    | 500ms | Fastest | Highest |
    | 1000ms | Default | Moderate |
    | 3000ms | Slowest | Lowest |
  </Step>

  <Step title="Start deletion">
    Click **Delete selected**. Deleo will:

    1. Search the selected channels for all your messages
    2. Delete each message with the configured delay between requests
    3. Show a live feed of deleted messages with progress percentage

    If Discord returns a `429` rate limit response, Deleo automatically waits 5 seconds before retrying.
  </Step>

  <Step title="Monitor progress and stop if needed">
    Watch the live deletion feed. Each entry shows the message content (truncated to 50 characters) and whether it was deleted or failed.

    Click **Stop** at any time to halt the process. Already-deleted messages cannot be recovered.

    <Warning>
      Deleted Discord messages are permanent. There is no undo.
    </Warning>
  </Step>
</Steps>
