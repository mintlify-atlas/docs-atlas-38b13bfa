## What is Deleo?

Deleo is an open source desktop application built with [Electrobun](https://blackboard.sh/electrobun) and React. It lets you delete your own Discord messages in bulk across any channel you have access to — DMs, group chats, and servers — with real-time progress tracking and automatic rate limit handling.

## Key features

## How it works

Deleo is built on top of the Discord API v9. It uses your personal user token to authenticate and then:

1. Fetches all your DM channels via `/users/@me/channels`
2. Searches each selected channel for your messages using `/channels/{id}/messages/search`
3. Deletes each message individually via `DELETE /channels/{id}/messages/{messageId}` with a configurable delay between requests

<Note>
  Deleo only deletes messages authored by you. It cannot delete messages from other users.
</Note>

## Getting started

<Steps>
  <Step title="Install prerequisites">
    Install [Bun](https://bun.sh/) on your system. Windows is the primary supported platform.
  </Step>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/kars1996/deleo.git
    cd deleo
    bun install
    ```
  </Step>
  <Step title="Get your Discord token">
    Use the included token-helper script or extract your token manually from Discord's DevTools.
  </Step>
  <Step title="Run Deleo">
    ```bash
    bun run dev
    ```
  </Step>
</Steps>

See the [Installation guide](/installation) for full setup instructions and the [Quickstart](/quickstart) to delete your first messages.
